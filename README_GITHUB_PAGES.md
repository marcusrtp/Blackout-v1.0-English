# GitHub Pages publication

This project is a static app: no build step, no npm dependency and no fragile module import.

## Folder to publish

Publish the contents of this `Blackout English/` folder, or copy it to your GitHub Pages root.

The folder contains:

- `index.html`
- `styles.css`
- `rooms.css`
- `game-polish.css`
- `dialogues.js`
- `puzzles.js`
- `script.js`
- `.nojekyll`
- `assets/`

## Browser cache

The files loaded by `index.html` use a `?v=...` parameter. Change that value after an update to force GitHub Pages and browsers to reload the latest version.

Current version: `english-version`.
